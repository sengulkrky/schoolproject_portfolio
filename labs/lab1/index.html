Lab 1 - Basics of HTML

Goals of the exercise

  • Setting up the basic structure of an HTML page.
  • Using semantic HTML tags such as <header>, <main> and <footer> to clarify the structure of your webpage.
  • Using a logical folder structure.
  • Adding images to your project.
  • Correctly publishing your website on a server.
Go to the solution

Description

I had to recreate a basic website only using HTML. This helped me learn the basic structure of a HTML-page and commonly used HTML-tags. In this exercise I also learned how to have a good folderstructure, as this is important for future projects.

Reflection

As I had some previous experience with HTML, this exercise was not that difficult. It had been a long time since I used HTML though, so it was really fun to work with it again. I had forgotten a lot of the basic elements, but doing this exercise really helped me remember those.
