Lab 3 - Advanced HTML

Goals of the exercise

  • Using tag-, class-, id- and pseudo-selectors.
  • Changing text and background colours.
  • Adding fonts from other sources.
  • Linking a stylesheet to your project.
  • Analysing CSS and debugging.
Go to the solution

Description

First of all I had to recreate a basic website only using HTML again. After that I had to link my own stylesheet to the project, but also a stylesheet from the internet. This time we also had to use a Google Font.

Reflection

In this exercise I learned a lot of new things. Some of them are the :root, ::after and ::before-elements. It was also the first time that I used a font from the internet. Until then, I had only used built-in fonts. This exercise was not the most difficult, but it was definitely challenging.
